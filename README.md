# 3-D PORTFOLIO

This is a 3-d personal portfolio.
It is deployed [here](https://shinchanthakur.github.io/3d-Portfolio/).

## Techs Used

1. React
2. Vite
3. Three.js
4. Tailwind css

## How to run

To run this project, follow these steps - 

1. Open a terminal and type the following commands - 
    - npm install
    - npm run dev
